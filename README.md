# Breakout

This is an attempt of recreating the popular arcade game Breakout. This version is in an early stage with updates coming out in the future. 
This project can only be compiled on Visual Studio for now.

# Version 1

In this version you'll have only a paddle moving, a ball and a grid of bricks. The ball can interact with the paddle (having alternating anngles depending where the ball touches, like in the original game). The ball can break bricks and receiving points. A player has 3 lives and if he runs out of lives, he loses the game. But if he breaks al the bricks, he will win. 

# Version 2

In this version I've added textures for the Ball, Paddle and Bricks. I've also added text for lives and score, and the following powerUps: ExpandPaddle, ShrinkPaddle, ExtraLife, MultiBall

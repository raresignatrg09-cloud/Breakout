# Breakout

This is an attempt of recreating the popular arcade game Breakout. This version is in an early stage with updates coming out in the future. 

# Version 1

In this version you'll have only a paddle moving, a ball and a grid of bricks. The ball can interact with the paddle (having alternating anngles depending where the ball touches, like in the original game). The ball can break bricks and receiving points.
